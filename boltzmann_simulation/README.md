# Boltzmann Simulation

Dieses Projekt visualisiert die **Boltzmann-Verteilung** bei unterschiedlichen Temperaturen.

## Voraussetzungen
- Python 3.12
- installierte Pakete aus der `requirements.txt`

## Installation
1. Virtualenv erstellen: `python -m venv .venv`
2. Aktivieren: `source .venv/bin/activate`
3. Pakete installieren: `pip install -r requirements.txt`
